It is a web application for shopping for book.

Features

    Site search and browse
    Book list by name
    Site security
    User can add book
    Admin can add,update and delete user
    Rest API

Technology Stacks

Backend

    Java 8
    Spring Boot 
    Spring Security
    Spring Data JPA
    Hibernate
    Oracle
    Maven

Frontend

    html
    javaScript
    css
    Bootstrap
    jQuery
